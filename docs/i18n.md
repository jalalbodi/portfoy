# i18n-ready Notes

The current interface is English-first, but the content is centralized in `data/profile.ts`, which makes an English/Turkish split straightforward.

Recommended next step:

```text
data/profile.en.ts
data/profile.tr.ts
lib/i18n.ts
app/[locale]/page.tsx
```

Keep URLs stable:

```text
/en
/tr
```

Do not machine-translate project outcomes. Rewrite them naturally and keep technical terms consistent.
